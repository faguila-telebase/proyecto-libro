# El Cosmos como Infierno / The Cosmos as Hell

## [ES] - Sinopsis
Los **agujeros negros** son infiernos. Los **cuásares** son infiernos fallidos.

## [EN] - Synopsis
**Black holes** are hells. **Quasars** are failed hells.

## [FR] - Synopsis
Les **trous noirs** sont des enfers. Les **quasars** sont des enfers ratés.

## [DE] - Zusammenfassung
**Schwarze Löcher** sind Höllen. **Quasare** sind gescheiterte Höllen.

## [RU] - Синопсис
**Черные дыры** — это ад. **Квазары** — это неудавшийся ад.

## [ZH] - 概要
**黑洞**是地狱。**类星体**是失败的地狱。
